## 1 设计哲学

### 1.1 设计语言

**「纸墨意境 · Scholarly Warmth」**

灵感来源于纸墨——纸张的温润质感与墨色的沉稳深邃。整个界面像一本精心编排的练习册，让学习者在数字环境中感受到纸质学习材料的亲近感。

| 关键词 | 诠释 |
|:------|:-----|
| 温润 | 暖色基底、柔和光影，不冰冷 |
| 沉稳 | 墨色为主、色彩克制，不浮躁 |
| 有序 | 清晰的信息层级、一致的节奏 |
| 专注 | 去除装饰干扰，内容即界面 |

### 1.2 桌面规范

本产品为 Windows 桌面应用（PySide6），所有交互遵循桌面平台规范：

| 原则 | 说明 |
|:----|:-----|
| **标准窗口** | 系统原生标题栏，不自绘 |
| **面板滚动** | 内容在面板内独立滚动，非整页滚动 |
| **选中即焦点** | 高亮条标记选中项，非 Web 的 outline 边框 |
| **右键菜单** | 列表项支持右键上下文菜单 |
| **键盘操作** | Tab 切换焦点，Enter 确认，Esc 取消 |
| **快捷键** | Ctrl+Enter 快速生成，Delete 删除 |
| **原生对话框** | 文件选择使用 QFileDialog |
| **持久状态** | 记住窗口大小、上次路径、折叠状态 |

### 2.2 字体系统

| 用途 | 字体 | 权重要求 |
|:----|:-----|:---------|
| 英文正文 | Segoe UI / DM Sans | Regular 400 |
| 英文标题 | Segoe UI / DM Sans | Bold 700 |
| 中文正文 | Microsoft YaHei / Noto Sans SC | Regular 400 |
| 中文标题 | Microsoft YaHei / Noto Sans SC | Medium 500 / Bold 700 |
| 音标 | Consolas / JetBrains Mono | Regular 400 |

**字号层级：**

| 名称 | 大小 | 字重 | 行高 | 用途 |
|:----|:----|:-----|:-----|:------|
| H1 | 24px | Bold | 32px | 页面主标题 |
| H2 | 18px | Medium | 26px | 区块标题 |
| H3 | 15px | Medium | 22px | 卡片标题 |
| Body | 14px | Regular | 20px | 正文、句子 |
| Small | 12px | Regular | 18px | 辅助信息、标签 |
| Caption | 11px | Regular | 16px | 角标、底部提示 |
| Button | 14px | Medium | — | 按钮文字 |
| Phonetic | 13px | Regular | 18px | 音标 |

### 2.3 间距体系

```
  间距 (4px 基准):
  xs = 4px    组件内间距（极小）
  sm = 8px    组件内间距（小）
  md = 12px   组件内间距（中）
  lg = 16px   卡片内边距
  xl = 24px   组件间距
  xxl = 32px  大区块间距
  page = 40px 页面边距
```

### 2.4 圆角体系

```
  sm  = 4px    标签、chip、小元素
  md  = 6px    按钮、输入框
  lg  = 8px    卡片、面板
  xl  = 12px   大卡片、模态框
  full = 999px 头像圈、开关
```

### 2.5 层级（Z-order）

桌面应用通过**边框 + 背景色差**区分层级，不依赖大面积阴影。

| 层级 | 样式 | 适用 |
|:----|:-----|:-----|
| 0 底层面板 | 背景 #F5F0E8, 无边框 | 页面背景 |
| 1 表面面板 | 背景 #FFFFFF, 1px #D8D2C8 边框 | 卡片、输入区 |
| 2 弹窗 | 背景 #FFFFFF, 1px #C4BEB4 边框, 阴影 | 对话框 |
| 3 全屏遮罩 | 半透明 overlay | 模态操作 |

## 10 后端 API 接口

### 10.1 接口总览

所有接口均为同步/异步调用（同一进程内），不涉及 HTTP 通信。

```
class StudentService:
    def get_all()                -> List[Student]
    def get_by_id(id)            -> Student
    def add(name)                -> Student
    def delete(id)               -> bool

class ArchiveService:
    def list(filter)             -> List[ArchiveSummary]
    def get_detail(session_id)   -> ArchiveDetail
    def delete(session_id)       -> bool
    def export(session_id, path) -> bool
    def import_archive(path)     -> bool

class GenerateService:
    def generate(req)            -> GenerationResult
    def cancel()                 -> bool

class ConfigService:
    def get_all()                -> Config
    def save(config)             -> bool
    def verify_api_key(key)      -> VerifyResult

class StatsService:
    def get_overview()           -> StatsOverview
    def get_by_level()           -> List[LevelStat]
    def get_by_weakness()        -> List[WeaknessStat]
    def get_trend(days)          -> List[DayStat]
```

### 10.2 核心数据结构

```python
@dataclass
class GenerateRequest:
    student: str
    level: str                    # KET/PET/中考/高考/雅思
    weakness_types: List[str]     # pronunciation/collocation/structure/implicature
    sentence: str                 # 原句
    details: str                  # 补充信息
    exercise_count: int           # 3/5/10/自定义

@dataclass
class GenerationResult:
    session_id: str
    diagnosis: Dict               # 按类型分组的诊断分析
    exercises: List[Exercise]     # 生成的练习句列表
    audio_paths: List[str]        # 音频文件路径列表

@dataclass
class Exercise:
    index: int
    weakness_type: str
    sentence: str
    translation: str
    target: str                   # 目标单词/搭配/结构
    explanation: str
    phonetic: str                 # 音标（发音类型）
    audio_file: str               # 音频文件名
    audio_duration: float         # 秒

@dataclass
class ArchiveSummary:
    session_id: str
    student: str
    level: str
    weakness_types: List[str]
    sentence: str                  # 原句预览
    created_at: str                # ISO datetime
    path: str                      # 文件系统路径
    exercise_count: int
    total_duration: float

@dataclass
class ArchiveDetail:
    summary: ArchiveSummary
    diagnosis: Dict
    exercises: List[Exercise]

@dataclass
class Config:
    api_key: str
    voice: str                     # 美式英语/英式英语
    default_count: int
    default_speed: str             # normal/slow
    theme: str                     # light/dark
    default_save_dir: str
```

### 10.3 存档筛选参数

```python
@dataclass
class ArchiveFilter:
    student: Optional[str] = None
    level: Optional[str] = None
    weakness_type: Optional[str] = None
    date_from: Optional[str] = None   # YYYY-MM-DD
    date_to: Optional[str] = None     # YYYY-MM-DD
    keyword: Optional[str] = None     # 全文搜索
    page: int = 1
    page_size: int = 20
```
